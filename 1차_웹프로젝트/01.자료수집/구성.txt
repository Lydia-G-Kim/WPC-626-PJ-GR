🎬 EMK 뮤지컬 홈페이지 리디자인 구성안

0. 인트로 애니메이션 흐름 (Splash → Main)
[1단계: Splash 화면]

전체 화면 레이아웃: 무대 커튼 + 정중앙 EMK 네온 로고

효과: 로고 테두리는 애니메이션 네온 깜빡임(CSS 애니메이션), 살짝 흔들리는 효과 추가

유저 액션: 로고 클릭

[2단계: 커튼 오픈 애니메이션]
로고가 위로 올라가며 점점 작아짐 (Transform: translateY(-100%) + scale(0.6))

좌우 커튼이 열리며 배경은 완전 블랙

커튼 뒷 배경엔 스포트라이트 1줄기가 중앙으로 떨어짐 (CSS radial-gradient 애니메이션)

무대 위로 공연 중인 뮤지컬 짧은 하이라이트 영상 재생

약 5초 정도 스포트라이트 점진적으로 확장 → 영상 약 1~2초 재생되고 영상 사라지고 메인 사이트로 자연스럽게 전환 (fade-in 효과)

🌆 1. 메인 홈페이지 레이아웃 구성
헤더 & 네비게이션
로고는 좌측 상단 고정, 스크롤 내려도 상단에 고정된 채 네온으로 깜빡임 유지

우측 상단 메뉴 아이콘: 햄버거 아이콘이 아닌, 동그란 네온 버튼 + 누르면 iOS 메시지 꾹 누르기 같은 "미리보기 팝업 카드" 형태

🎫 2. 배너 인터랙션: 티켓 파노라마
배경 구성:
배너 섹션 전체가 뮤지컬 티켓 일러스트(전체적으로 흩뿌려진 느낌)로 채워짐 (PNG + 투명 배경 + 3D 퍼스펙티브)

마우스 휠 스크롤 시 티켓들이 살짝 회전하며 움직이고, 그 중 하나가 전면으로 튀어나옴

동적 콘텐츠:
튀어나온 티켓이 확대되며 → 그 공연의 짧은 영상 or 포스터 이미지가 자동 전환됨

다음 스크롤 → 다음 티켓으로 슬라이드 전환

하단엔 예매 버튼(네온 + Hover 시 빛 확장)

✅ 효과 예시:

티켓이 튀어나올 때마다 살짝 종이 떨어지는 소리 + 네온 사운드

마우스 Hover 시 부드러운 shadow와 glow

🌃 3. 전체 페이지 톤앤무드
배경 :
다크한 느낌, 곳곳에 네온 선 or 점선

포인트 색:
레드(#FF3C3C), 퍼플(#C74AFF), 옐로우(#FFE400), 전기블루(#00F0FF) 등 네온 느낌

폰트 :
메인 타이틀: Broadway/Playbill 스타일 + 네온 glow, 본문은 Noto Sans or Pretendard

애니메이션:
마우스 따라다니는 spotlight, 커서 오버 시 티켓 흔들림 등 마이크로 인터랙션 적극 활용

💡기술 구현 팁
인트로 애니메이션:
GSAP (GreenSock)이나 Lottie + After Effects 연동으로 구현 가능
→ 배너 티켓 인터랙션도 GSAP ScrollTrigger로 자연스럽게 제어 가능

네온 효과:
CSS text-shadow + box-shadow 조합 + @keyframes를 이용한 깜빡임 애니메이션
(필요 시 WebGL 기반 네온 플러그인 사용 고려)

메뉴 미리보기:
Framer Motion or React Spring 라이브러리로 iOS-style 팝업 인터랙션 구현 용이
